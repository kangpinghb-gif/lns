# Life Navigation System — Product Definition

> Version: 1.0 | Status: Product Core Definition (Frozen)

## 1. Product One-Line Definition

LNS 是一个基于时间与结构模型的人生决策导航系统。

## 2. Core Product Reframe

我们不做：算命、预测、神秘解释。

我们做的是：**人生导航系统（Life Navigation System）**。

类比：
- Google Maps（路径）
- Apple Health（状态）
- Notion AI（决策）

## 3. Core Value Proposition

用户真正买的不是「命理」，而是**「降低人生不确定性」**。

系统提供三件事：

1. **我现在在哪？** — 人生阶段、能力结构、风险结构
2. **我该往哪走？** — 时间趋势、机会窗口、发展方向
3. **我现在该做什么？** — P0-P3 行动、决策优先级、风险规避

## 4. Product Positioning

**We ARE NOT：** 占卜工具、命理分析工具、心理测试工具。

**We ARE：** 人生决策操作系统（Life Decision OS）。

## 5. Target Users

### Core Users
- 迷茫期人群（18–35）
- 职业转型人群
- 创业者
- 高不确定性职业（自由职业 / 投资 / 创作者）

### Secondary Users
- 管理者
- 决策频繁人群
- 长期规划人群

## 6. Core Use Cases

| Use Case | 用户问题 |
|----------|----------|
| 人生方向 | 我适合做什么？ |
| 职业决策 | 我要不要换工作？ |
| 时间窗口 | 现在适合创业吗？ |
| 风险判断 | 我现在是不是在错误路径？ |

## 7. Product Architecture Summary

```
Calendar Engine → BaZi Engine → State Engine → Knowledge Graph
    → Time Engine → Decision Engine → Prompt Engine → AI Output
```

## 8. Core Differentiation

### vs 传统命理

| 传统命理 | LNS |
|----------|-----|
| 玄学解释 | 结构化系统 |
| 吉凶判断 | 行动建议 |
| 命运预测 | 时间导航 |
| 模糊表达 | P0-P3 决策 |

### vs AI 助手

| AI 助手 | LNS |
|---------|-----|
| 通用回答 | 结构化人生系统 |
| 无状态 | 有人生模型 |
| 无时间系统 | 时间驱动 |
| 无决策体系 | P0-P3 系统 |

## 9. Core Product Loop

```
输入出生信息 → 生成人生结构 → 生成时间系统
    → 生成状态系统 → 生成决策系统 → 用户执行 → 反馈回流
```

## 10. Monetization Model

| Tier | 包含功能 |
|------|----------|
| **Free** | 人生总览、当前状态、基础建议 |
| **Pro** | 时间分析（T0-T3）、AI问答、决策系统 |
| **Premium** | 年度人生报告、深度职业分析、多路径模拟 |
| **Enterprise** | 批量分析、团队决策模型、API调用 |

## 11. Product Metrics

核心指标：决策采纳率、用户回访率、行动完成率、人生路径稳定性。

## 12. Product Philosophy

系统不是为了「解释人生」，而是帮助用户做出更少错误决策。

## 13. Emotional Positioning

| 不提供 | 提供 |
|--------|------|
| 安慰 | 清晰 |
| 恐吓 | 结构 |
| 宿命 | 行动路径 |

## 14. System Identity

LNS is: A structured decision system over time-based human modeling.

## 15. Product Boundary

### In Scope
状态建模、时间分析、决策生成。

### Out of Scope
预测具体事件、宗教解释、情绪操控。

## 16. Final System Definition

LNS 是一个把「人生不确定性」转化为「结构化决策路径」的系统。

## 17. Extension (Future Vision)

- 全球人生导航系统
- AI 人生模拟器
- 多人生路径对比系统
- 实时人生操作系统（Life OS）
