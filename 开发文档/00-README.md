# Life Navigation System

> Version: 1.0 | Status: Frozen

## 1. 产品定义

LNS 是一个基于八字与时间建模的人生决策系统。不预测未来，而是将用户人生转化为「状态 + 时间 + 行动」的结构化导航系统。

## 2. 核心问题

LNS 唯一回答的问题：**如何帮助用户做出更好的下一步决策？**

## 3. 不是算命系统

- **不预测未来** — 不宣称能预见尚未发生的事件
- **不宿命论** — 不承认人生轨迹已被预先注定
- **不制造焦虑** — 不以恐吓、危言或吉凶预言驱动用户
- **不绝对判断** — 不给出「好/坏」「吉/凶」的二元结论
- **不做无法验证的神秘解释** — 所有输出可溯源、可检验、可质疑

## 4. 核心方法

1. **理解当前状态** — 基于用户出生时间与当下时间，构建结构化的人生状态模型
2. **理解变化趋势** — 分析时间维度上的能量流动与阶段性偏移
3. **生成行动建议** — 将抽象趋势转化为可落地、可执行的具体行动指引
4. **支持人生决策** — 最终服务于用户在职业、关系、健康、成长等领域的实际选择

## 5. 核心理念

> 命理不是命运，人生不是定数。八字的本质是出生时间与天时地利相互作用下形成的倾向性结构。LNS 利用这种结构建模，但不被困在结构里。

## 6. 产品定位

| 类比 | 说明 |
|------|------|
| **Apple Health** | 像监测身体状态一样监测人生状态 |
| **Google Maps** | 像导航路线一样导航人生路径 |
| **Notion** | 像组织知识一样组织决策逻辑 |
| **不是算命、占卜、风水工具** | 拒绝所有玄学包装与神秘主义定位 |

## 7. 与传统命理区别

| 维度 | 传统命理 | LNS |
|------|----------|-----|
| 输出 | 大师解释 → 用户接受 | 状态建模 → 用户决策 |
| 角色 | 权威解读 | 辅助工具 |
| 验证 | 难以验证或事后归因 | 可追溯、可检验 |
| 立场 | 被动接受结论 | 主动参与决策 |

## 8. 与 AI 区别

| 维度 | 普通 AI Chat | LNS |
|------|--------------|-----|
| 本质 | 文本生成 | 结构化状态 + 时间 + 决策系统 |
| 输出 | 概率性文字应答 | 基于模型的确定性建议框架 |
| 架构 | 通用语言模型 | 领域专家系统 + 时间分析引擎 |

## 9. 最终承诺

> LNS 不承诺预测未来。
>
> LNS 承诺：帮你理解现在、帮你规划下一步。

## 10. 法律声明

LNS 是一套基于古典时间建模与行为学映射的理性决策辅助工具。系统输出的所有建议、风险提示及趋势分析均基于结构化模型推演，**仅供用户参考，不构成任何具有法律效力的决策指令或投资建议。最终决定权与责任完全归用户所有。**
