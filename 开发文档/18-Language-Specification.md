# Life Navigation System — Language Specification

> Version: 1.0 | Status: Frozen

## 1. Purpose
定义整个产品的命理→产品语言翻译标准。任何 Prompt、AI 输出、页面文字都不能自行翻译，全部调用本规范。

## 2. 十神→产品语言
- 正财 → 资源管理能力（稳定的财富积累与财务管理）
- 偏财 → 机会型资源获取（非固定收入与风险投资能力）
- 正官 → 规则执行能力（秩序维护与体制内发展能力）
- 七杀 → 压力转化力（在对抗与挑战中激发行动力）
- 正印 → 学习与支持能力（稳定的知识积累与外部资源接收）
- 偏印 → 独立思考能力（非主流学习路径，擅长跨界整合）
- 比肩 → 自主执行力（独立决策与自我驱动）
- 劫财 → 资源竞争能力（高频行动与机会捕捉）
- 食神 → 持续输出能力（稳定创造与内容生产）
- 伤官 → 表达与创新能力（挑战规则与突破性表达）

## 3. 五行→产品语言
木→成长/学习/突破, 火→表达/影响/传播, 土→稳定/积累/执行, 金→规则/效率/管理, 水→思考/智慧/连接

## 4. 刑冲合害→产品语言
冲→变化, 刑→磨练, 害→干扰, 克→制约, 合→协同

## 5. 吉凶转换
全部废除吉凶概念，改为：机会、风险、趋势、窗口、变化、结构、节奏。

## 6. AI语言原则
禁止：命中注定、天命、大凶、必然、一定、一定发财、一定离婚。
统一：当前更适合…、建议…、可能出现…、值得关注…、建议提前准备…

### 补充：正官
正官 → 规则执行能力、秩序维护能力
behavior: 遵守规则、计划执行、稳健决策
career: 公务员、管理、法务
risk: 灵活性不足

### 补充：七杀
七杀 → 挑战驱动力、突破执行力
behavior: 高压力决策、竞争导向、不畏惧冲突
career: 创业、项目管理、危机处理
risk: 冲动、压力过载
advice: 适合高压环境，但需要结构缓冲

### 补充：正印
正印 → 学习吸收能力、知识支撑力
behavior: 深度研究、知识积累、系统化思考
career: 教育、研究、顾问
risk: 行动迟缓
advice: 适合知识密集型路径，注意与实践结合

### 补充：偏印
偏印 → 独特思考能力、非常规视角
behavior: 独立研究、跨界思考、非主流路径
career: 科研、技术开发、创新行业
risk: 与社会脱节
advice: 适合创新探索，但需要外界反馈机制

## 7. 神煞系统 → 现代语言对照

### 桃花

### 文昌

### 红艳（低频/特殊场景）

→ 结构性困局信号
译为：当前面临的结构性约束较强，不宜硬闯，建议等待时机或寻找侧翼突破
