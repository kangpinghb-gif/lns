# AGENTS.md

## 项目概况

LNS 是一个人生导航系统，后端为 Python 服务，前端页面位于 `static/`。

当前前端主结构已经调整为多页面静态骨架：

- `home.html`：品牌首页
- `create.html`：出生信息输入页
- `kline.html`：人生 K 线总览页
- `analysis.html`：命理透视页
- `year-detail.html`：流年详情页

兼容入口保留：

- `index.html`：跳转到 `home.html`
- `dashboard_static.html`：跳转到 `kline.html`

## 工作约定

- 默认用中文沟通。
- 修改前端页面时优先保持现有静态路径、API 路径和共享资源文件稳定，避免影响后端路由与后续接接口。
- 简单优先，优先交付可运行页面，不为单页静态界面引入不必要依赖。
- 涉及删除、部署、数据库写入、环境变量或批量高风险修改前，先说明影响范围并等待确认。

## 验证

- 静态页面修改后至少检查 HTML 结构和关键脚本是否仍可解析。
- 如启动服务验证，优先使用项目已有启动方式。
